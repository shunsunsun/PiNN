PiNNboardÂ¶

PiNNboard is a Tensorboard plugin to visualize atomic neural networks by showing activations for atoms and pairs. PiNNboard currently provides visualization of the PiNet neural network, but it also provides tools to create the visualization for different network architectures.

Warning

Please note that PiNNboard relies on experimental features of Tensorboard, and is not considered stable at this point.
